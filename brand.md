# DC Kirrberg e.V. Brand Guidelines

## Club Identity

**Full Name:** DC Kirrberg e.V.
**Tagline:** Die Sterne vom Tussberch
**Slogan:** GOOD DARTS in Homburg an der Saar

## Color Palette

### Primary Colors

| Color         | Hex       | Usage                     |
| ------------- | --------- | ------------------------- |
| Heraldic Red  | `#C8313E` | Accents, highlights, logo |
| Dark Burgundy | `#2D0A0A` | Backgrounds, gradients    |
| White         | `#FFFFFF` | Text, logo elements       |

### Design Principle

Black as the base color, red (`#C8313E`) for highlights and borders.

### Gradient

Background gradient: Dark burgundy (`#1A0505`) to medium dark red (`#4A1515`)

## Logo

Circular badge featuring:

- Outer ring: "DC KIRRBERG e.V." (top) and "DIE STERNE VOM TUSSBERCH" (bottom)
- Dartboard pattern as background
- Kirrberg coat of arms center (red cross, red lion on shield)
- Beer mugs and darts flanking the shield

## Typography

- **Headlines:** Bold sans-serif, italic style
- **Subheadlines:** All caps, increased letter spacing
- **Body:** Clean sans-serif

## Design Elements

- Red accent stripe/bar for visual interest
- Dark gradient backgrounds create depth
- Logo placement: top-right corner or centered-left
- Ample white space around text
